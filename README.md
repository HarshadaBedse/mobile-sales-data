# mobile-sales-data
Motorola Mobile Sales Dashboard

An interactive Power BI dashboard designed to analyze mobile sales performance across regions, products, and customer behavior. The dashboard provides a clear overview of key metrics through KPI cards, geographic mapping, and trend visualizations.

It enables users to explore sales by city, mobile model, month, payment method, and weekday using dynamic filters. Visual insights include total sales distribution, monthly quantity trends, customer ratings, and payment method usage, supporting data-driven decision-making.

Features

KPI cards for total sales, quantity, and transactions

City-level sales map visualization

Sales trends by month and day

Mobile model performance comparison

Payment method distribution analysis

Customer ratings breakdown

Interactive filters for detailed exploration

Purpose

Built for business intelligence, sales analysis, and data visualization practice, this dashboard demonstrates how interactive reporting can uncover trends, monitor performance, and support strategic decisions.
